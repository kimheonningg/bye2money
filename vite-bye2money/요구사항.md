### 1. 헤더 영역

상단의 헤더 부분은 가계부의 모든 화면에서 상단 중앙에 표시한다.

1. 헤더의 가장 왼쪽에는 로고를 표시한다. 로고를 클릭하면 메인 화면으로 이동한다.

2. 헤더 중앙에 현재 연월을 표시한다. 좌우 화살표를 클릭하면 이전달과 다음달로 이동하고, 하단 내역 리스트 또한 선택한 월에 해당하는 내용으로 함께 변경되도록 한다.

3. 헤더의 가장 오른쪽에는 내역/달력/통계 탭을 표시한다. 각 탭을 누르면 각 화면으로 이동한다.

### 2. 새로운 내역 입력

헤더 아래에는 수입 지출 내역을 입력할 수 있는 바가 있다.

1. 날짜 인풋 박스에는 기본으로 오늘 날짜가 입력되어 있으나, 수정할 수 있도록 한다.

2. 금액을 입력하는 인풋 영역 앞에 있는 [마이너스 버튼]은 지출을 의미하며, 이 버튼을 클릭하면 [플러스 버튼]으로 바뀌며 수입 내역을 입력할 수 있게 된다. [마이너스 버튼] 상태에서는 [분류 셀렉트 박스]에서 ‘지출 카테고리 드롭다운 패널'이 나타나고, [플러스 버튼] 상태에서는 ‘수입 카테고리 드롭다운 패널’이 나타나도록 한다. 이 버튼은 지출을 기본값으로 한다.

3. 금액을 입력할 땐, 자동으로 세 자리마다 쉼표가 표현되도록 한다. 각 탭을 누르면 각 화면으로 이동한다.

4. 내용 입력 칸에는 해당 내역에 대한 설명을 적을 수 있다. 최대 32자까지 작성 가능하며, 라벨 좌측 끝에 글자 수가 카운트 되도록 한다.

5. 결제수단을 입력하는 셀렉트 박스를 클릭하면 결제수단 드롭다운 패널이 나타난다.

6. 항목마다 우측에 표시된 [X 버튼]을 누르면 배경이 #000000, 40%로 딤처리되며 ‘해당 결제 수단을 삭제하시겠습니까?’ 알럿창이 뜬다.
   최종적으로 [삭제 버튼]을 누르면 해당 결제수단이 삭제되고, 드롭다운 패널에서도 사라진다.
   삭제한 결제수단으로 작성된 수입지출내역은 빈칸으로 남도록 한다.

7. 드롭다운 패널 마지막의 [추가하기]항목을 누르면 ‘추가하실 결제 수단을 입력해주세요'라는 모달창이 뜨도록 한다.
   결제수단의 이름을 입력하고 최종적으로 [추가 버튼]을 누르면 드롭다운 패널에 결제수단이 추가되도록 한다.

8. 분류 셀렉트 박스를 클릭하면 카테고리 드롭다운 패널이 나타나는데, 수입인지 지출인지에 따라 나타나는 목록이 달라지도록 한다.

- 수입 카테고리: 월급, 용돈, 기타수입
- 지출 카테고리: 생활, 식비, 교통, 쇼핑/뷰티, 의료/건강, 문화/여가, 미분류

9. 모든 내용을 입력해야 [확인 버튼]이 활성화 된다. [확인 버튼]을 누르면, 아래 내용으로 유효성 검증을 하며, 이 부분은 스스로 기획한다. 유효성 검증이 통과되면, 수입지출 내역이 저장되고 아래의 내역 목록에 입력한 내용이 추가된다.

### 3. 수입 지출 내역 목록

1. 수입지출 리스트 상단에는 내역의 총 건수와 해당월의 수입액과 지출액이 각각 표시되도록 한다.
   표시된 수입액과 지출액 앞에 배치한 체크박스를 통해 리스트에 표시되는 수입 내역과 지출 내역에 대한 필터링을 제어할 수 있도록 한다.
   수입액과 지출액 모두 보여지는 것을 기본으로 한다.

2. 수입지출 리스트는 입력된 ‘일자'를 기준으로 최신순으로 정렬되도록 한다.
   같은 일자 안에서 여러 내역의 경우, 생성된 시간을 기준으로 정렬되도록 한다.

3. 표시된 수입액과 지출액 앞에 배치한 체크박스를 통해 리스트에 표시되는 수입 내역과 지출 내역에 대한 필터링을 제어할 수 있도록 한다.
   일별 리스트 상단에는 ‘일자’, ‘요일', ‘일별 수입액'과 ‘일별 지출액'을 각각 종합하여 나타내도록 한다.
   수입액과 지출액 모두 보여지는 것을 기본으로 한다.

4. 지출 내역은 빨간색, 수입 내역은 초록색으로 표시한다.

5. 수입지출의 각 내역에 마우스를 올리면, 배경이 #FFFFFF색으로 변경되며 오른쪽에 [삭제 버튼]이 생기도록 한다.
   [삭제 버튼]을 누르면 수입지출 내역을 삭제할 수 있는 알럿창이 뜬다.
   알럿창의 [삭제 버튼]을 누르면 해당 내역이 1초 지연 뒤에 삭제되도록 하고, [취소 버튼]을 누르면 알럿창만 꺼지도록 한다.

6. 수입지출의 각 내역에 마우스를 올린 상태로 [삭제 버튼]을 제외한 다른 영역을 클릭하면, 해당 내역의 내용이 입력바에 채워지고, 내용을 수정할 수 있도록 한다.

7. 기존 내용에 수정사항이 생겼을 때, [확인 버튼]이 활성화 되도록 한다. [확인 버튼]을 누르면 즉시 입력바는 비워지고, 기존 내용에는 수정사항이 반영된다. 선택되었던 수입지출 내역은 마우스로 내역 입력바가 아닌 다른 영역을 클릭하거나 수정을 마치고 [완료 버튼]을 눌렀을 때, hover/selected 상태에서 해제되도록 한다.

### 4. 달력 화면

1. 헤더의 [달력 탭]을 선택하면 일별 내역 통계를 캘린더로 확인할 수 있는 화면으로 이동한다.

2. 오늘 날짜에 해당하는 날은 배경색을 다르게 표시한다.

3. 수입, 지출, 일별 합계는 각각 다른 색으로 표시한다.

4. 달력 하단에는 해당 월의 총 수입, 총 지출, 총합을 표시한다.

### 5. 통계 화면

1. 헤더의 [통계 탭]을 선택하면 카테고리별 지출 내용을 확인할 수 있는 페이지로 이동한다.

2. 해당 월의 총 지출금액을 표시한다.

3. 해당 월의 내역 데이터를 기반으로 가장 비율이 큰 카테고리순으로 정렬한다. 각 리스트 아이템에는 카테고리명, 비율, 해당 카테고리의 지출합을 표시한다.

4. 원한다면, 통계 화면이 불러와질 때 그래프와 각 카테고리 리스트에 애니메이션을 적용해볼 수 있다.

5. 리스트 아이템을 클릭하면 해당 카테고리의 월별 소비추이 그래프와 상세 내역이 아래로 펼쳐진다.

6. 소비 추이 그래프는 해당 카테고리의 최근 6개월 지출 내역으로 그린다. 상세 내역은 메인에서 리스트로 나온 방식과 동일하게 표시한다.
